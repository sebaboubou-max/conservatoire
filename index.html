Le principe d'incertitude : M × C ≤ K  (K=4, r=−0.642, n=48, 0 violateurs)
Plus un opérateur mûrit lentement (M↑), moins il se compose facilement (C↓). Les opérateurs instantanés (M=0) sont les plus composables. Les opérateurs cross-session (M=3) opèrent seuls — comme le levain qui fait lever la pâte sans se mélanger.

Les 10 Paradoxales

Les paires les plus instables ET les plus composables. solve coagula × π (173.1), lichtung × fouille (170.2), essaim × ablation (166.6). Le paradoxe est le terreau des proto-zinzins.

L'Inversion du Signe (Eq.80)

Le sens de la contrainte dépend du niveau. Niv.1 : r=−0.685 (le temps contraint). Niv.2 : point fixe (M=2, C=2). Niv.3 : r=−0.113 (la contrainte se dissout). L'axe n'est pas une flèche — c'est un gradient.

Les Deux Monnaies (Eq.78)

Famille temporelle : M × C ≤ 4. Famille spatiale : A × R ≤ 9. Les deux sont conjuguées (r < 0). Optimiser dans le temps dégrade dans l'espace. On ne peut pas tout avoir.
